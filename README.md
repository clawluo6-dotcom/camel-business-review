# 骆驼商业本质

> 2026-06-04 构建 | AI 商业深度观察 · 科技产业速报

## 项目概述

一个采用 **web-db-user 三层架构** 的 AI 行业新闻网站，聚焦每日 AI 领域最新动态。深色科技风界面，支持后台内容发布。

## 技术架构

```
┌─────────────────────────────────────────┐
│              web 层 (index.html)          │
│   前端展示：新闻列表、详情、分类筛选、统计  │
│   Tailwind CSS CDN + Chart.js 图表       │
├─────────────────────────────────────────┤
│              db 层 (news.json)            │
│   数据存储：22条新闻、6个分类、时间线      │
│   内置数据兜底，无需服务器                  │
├─────────────────────────────────────────┤
│             user 层 (admin.html)          │
│   后台管理：增删改新闻、JSON导出            │
│   localStorage 持久化                     │
└─────────────────────────────────────────┘
```

## 文件说明

| 文件 | 用途 |
|------|------|
| `index.html` | 主站前端，新闻浏览 |
| `admin.html` | 后台管理面板 |
| `data/news.json` | 新闻数据库（22条，6分类） |
| `README.md` | 本文档 |

## 功能特性

- **6 大分类**：大模型 / 智能体 / 硬件算力 / 开源生态 / 政策监管 / 资本并购
- **深度速报**：精选头条 + 分类浏览 + 全文阅读
- **数据可视化**：Chart.js 统计图表（幻觉率对比、token 成本趋势）
- **时间线**：6月3-4日重大事件时间轴
- **响应式设计**：桌面/平板/手机自适应
- **后台管理**：可视化编辑发布，localStorage 持久化
